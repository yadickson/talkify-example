> Talkify Text to Speech Example

This application use AngularJS, Bootstrap and Html5

## Prebuild application

```bash
yarn install
```

## Run application

```bash
gulp serve
```

```bash
gulp serve:dist
```

## Test application

```bash
gulp test
```

```bash
gulp serve:test
```

## Build application

```bash
gulp build
```

## Dist application

```bash
gulp dist
```

## Clean project

```bash
gulp clean
```
